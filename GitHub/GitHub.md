## Git
프로그램 등의 소스 코드 관리를 위한 분산 버전 관리 시스템이다. Git의 작업 폴더는 모두 기록하고 있어서 추적이 가능하고, 완전한 형태의 저장소이다.

## GitHub
Git을 호스팅해주는 웹 서비스이며, Git 저장소 서버를 대신 유지 및 관리해주는 서비스이다.
